# Assignment6_Excel_Data_Analytics
This is an end-to-end data analytics project done to explore Advanced EXCEL Functions.

In this project, the following concepts were practiced and explored:-

• Excel basics like SUM,AVERAGE,MIN,MAX etc..
• Advanced formulas like SUMIF,MAXIFS,CONDITIONAL FORMATTING
• Data cleaning using TRIM(),Removing Duplicates,Changing Data Types etc..
• Pivot tables to Summarise the dataset which has around 9995 rows and 21 columns, to findout Sales by Region,Category and Monthly Sales Trend.
• Dashboards to have an overall summary of the dataset.

The repository includes:-

1.Assignment-6.xlsx- The Excel Workbook with Tasks 
2.Assignment-6-documentation.pdf - Screenshots of Formulas, Outputs and Answers to the Conceptual Questions Asked.
